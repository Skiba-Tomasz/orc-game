## Running the project

- Clone repository
- Install pygame `pip install pygame`
- Run `main.py` located in `src` folder using `pytohon3`

## Controlls

- Movement - arrow keys
- Attack - `c`
- Exit game `q` / click at window bar
- Restart/finish - click (yes/no)
